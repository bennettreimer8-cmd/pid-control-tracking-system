# pid-control-tracking-system

Project Description :  This engineering project demonstrates a closed-loop position tracking system that automatically controls the position of a car on a tilting beam. One HC-SR04 sensor is placed on the end of the beam that continuously measures the distance from it and the car and another one is on a separate but parallel track that measures the distance between it and a wooden block. The block is used as a target distance so whenever the block is moved along the track there is an Arduino that compares the measurements and calculates the position error between the block and the car. A PID controller then processes the error and commands a servo motor to adjust the beam's angle moving the car left or right until its position matches the position of the block.
